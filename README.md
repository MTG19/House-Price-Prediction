# House Price Prediction – Advanced Regression Analysis

## Project Overview
This project predicts house prices using a structured real-world dataset. It demonstrates **feature engineering, preprocessing, multiple regression models, encoding strategies, feature selection, and hyperparameter optimization**, providing insights into predictive modeling for real estate pricing.

## Key Highlights
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering (derived features & log transformations)
- Regression models: Linear, Ridge, Polynomial
- Encoding strategies: OneHot, Target Encoding
- Hyperparameter tuning and feature selection
- Model evaluation and comparison

## Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Category Encoders

## Repository Structure
- `HousePricePrediction.ipynb` – Main notebook with full analysis
- `HousePricePrediction.xlsx` – Dataset
- `README.md` – Project description and insights

## Business Impact
- Predict house prices accurately for informed real estate decisions
- Identify key features influencing price
- Apply best regression practices and encoding strategies for strong model generalization
